# Dispatch System

I'm building this project for the logistics company I work at. 
The goal is to replace our current Excel-based workflow with a 
proper management system that saves time and reduces human error.

Currently a console app, planning to turn it into a Desktop application.

## What it does
- Login system with different roles (Dispatcher, HR, Operation, Admin)
- Add, view and delete drivers
- Create and manage loads
- Assign loads to drivers
- Track load status in real time

## What's coming
- WPF Desktop UI
- Database (so data doesn't disappear on restart)
- Multi-user support

## Built with
- C# .NET
